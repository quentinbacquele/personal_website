A high-impact, "Ethereal Academia" portfolio website designed for a PhD researcher in Bioacoustics and AI.

## Tech Stack
- **Framework:** React (Vite) + TypeScript
- **Styling:** Tailwind CSS (v3)
- **Motion:** Framer Motion
- **Icons:** Lucide React

## Design System
- **Fonts:**
  - *Headings:* Instrument Serif (Editorial, Academic)
  - *Body:* Satoshi (Modern, Geometric)
  - *Mono:* JetBrains Mono (Data, Code)
- **Colors:**
  - Background: Obsidian (`#0a0a0a`)
  - Surface: Charcoal (`#121212`)
  - Accent: Acid Green (`#d4ff00`)
  - Text: Pale Sand (`#e3e3e3`)

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run development server:**
    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```

## Customization
- **Content:** Edit the components in `src/components/` (Hero.tsx, Research.tsx, About.tsx, etc.).
- **Research Data:** Update the `publications` array in `src/components/Research.tsx`.
